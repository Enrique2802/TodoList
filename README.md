# To Do List

Une page web simple de gestion de tâches en faite en HTML, CSS et JavaScript.

## Objectif

Projet de consolidation pour pratiquer :
- Les bases du JavaScript (variables, tableaux, fonctions)
- La manipulation du DOM
- Le travail en équipe

## Fonctionnalités

- Ajouter une tâche
- Afficher la liste des tâches
- Marquer une tâche comme complétée
- Supprimer une tâche
- Filtrer les tâches (Toutes/Actives/Complétées)
- Statistiques en temps réel

## Technologies

- HTML
- CSS
- JavaScript Vanilla
